# 🚨 AI-Based Suspect Identification System

## 🔍 Features
- Real-time face detection
- Face recognition using AI
- Police alert system
- React dashboard

## 🛠 Tech Stack
- React.js (Frontend)
- Spring Boot (Backend)
- Python (AI Module)

## 🚀 How to Run
1. Start Spring Boot backend
2. Start React frontend
3. Run Python AI module
